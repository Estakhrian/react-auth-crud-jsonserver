یک پروژه صفحه لاگین و لیست کاربران رو داریم 
پکیج های استفاده شده در این پروژه : 
react-router , react-bootstrap , json-server , axios 

در صفحه اصلی با کلیک روی دکمه login page  وارد صفحه لاگین میشیم 
باید اطلاعات رو وارد کنیم در صفحه لاگین و اگر فیلدی رو خالی بزاریم ذخیره اطلاعات در json server انجام نمیشه 
با کلیک روی دکمه users  در صفحه لاگین یا دکمه users list  در صفحه اصلی وارد صفحه ای میشیم که لیست کاربران اونجا قرار داره و این عملیات ها و جا به جایی بین صفحات با پکیج react-router انجام میشه 

به محض register کردن فرم اطلاعات کاربر مستقیم در json server ذخیره میشه توسط متد POST در Axios 
و موقع ران شدن صفحه مربوط به لیست کاربران اطلاعات از jsonserver میاد گرفته میشه توسط axios و متد GET 
در لیست کاربران با کلیک روی آیکون های ویرایش و حذف میتونیم کاربری رو از json server حذف کنیم یا  آپدیت بکنیم که اینکار رو با متد های PUT , DELETE انجام میدیم

در این پروژه تمامی عملیاتهای crud پیاده سازی و اجرا شده است
